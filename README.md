## About App
Ushbu project Laravelda bir necha fayllarni birdan yuklash(multiple file uploading) ni o'rganish uchun amaliy misol sifatida qurildi bu yerda rasmlarni yuklash ko'rsatilgan 

## O'rganilgan narsalar 
- Rasmlar uchun request yaratish va rule berish
- Fayllarni saqlashing <b>storeAs</b> va <b>Storage</b> metodlari
- Image resizing
- Unique filename yaratish
- Faylar meta datasni db ga saqlash
- Formdagi bir necha fayllarni yuborishning <b>synchronous</b> va <b>asynchronous</b> usullari
- Js yordamida asynchronous usuli bilan faylarni ketma-ket yuborish
